23.4 Cluster Annotations

In Dereck's paper, they used the imputed gene score to assign cluster annotations. The process is:

  1. Add a list of marker genes for each cell type.
  2. Get the imputed gene score for each marker gene.
  3. Plot each imputed gene score.
  4. Check the increase/decrease of the imputed gene score for each cell type.
  5. Manually assign the cluster annotations based on the imputed gene score.
